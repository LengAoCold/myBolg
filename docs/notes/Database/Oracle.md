#### SELECT 排序 ORDER BY

`默认为ASC升序`

##### ASC 按升序排序

##### DESC按降序排序

```sql
-- 按照字母顺序按升序排列
SELECT name,address,credit_limit
FROM customers
ORDER BY name ASC;

-- 按多个列排序行
SELECT first_name, last_name
FROM contacts
ORDER BY first_name, last_name DESC;

-- 按列位置排序行
SELECT name, credit_limit,address
FROM customers
ORDER BY 2 DESC, 1;		-- 数字代表查询列位置
-- 相当于
SELECT name, credit_limit,address
FROM customers
ORDER BY credit_limit DESC, name;

-- 按函数或表达式排序
SELECT customer_id, name
FROM customers
ORDER BY UPPER( name );
```

## 函数方法

#### ROUND

round：对某个值进行四舍五入
语法：round（number,decimals）
参数：
 number：要进行四舍五入的数值
 decimals：指明需保留小数点后面的位数。可选项，忽略它则截去所有的小数部分，并四舍五入。如果为正数则表示从小数点右边开始的位置进行四舍五入，如果为负数则表示从小数点开始左边的位数，相应整数数字用0填充，小数被去掉。



#### TRUNC

trunc：用于截取时间或者数值，返回指定的值

语法：trunc(number,[decimals])

参数：

​	number：指需要截取的数字

​	decimals：截取位数，可选参数，如果参数是负数表示从小数点左边截取。注意这里的截取都不做四舍五入。直接舍掉



#### NVL

Nvl：用于截取时间或者数值，返回指定的值

语法：nvl(number,[decimals])

参数：

​	number：指需值

​	decimals：如果表达式1为空值，NVL返回值为表达式2的值，否则返回表达式1的值。 该函数的目的是把一个空值（null）转换成一个实际的值。其表达式的值可以是数字型、字符型和日期型。但是表达式1和表达式2的数据类型必须为同一个类型。

## 比较运算符

| 运算符                | 描述                                                         |
| --------------------- | ------------------------------------------------------------ |
| IN                    | 等于值列表中的任何值                                         |
| ANY/SOME/ALL          | 将值与列表或[子查询](http://www.yiibai.com/oracle/oracle-subquery.html)进行比较。它必须以另一个运算符(例如：`=`，`>`，`<`)作为前缀。 |
| NOT IN                | 不等于值列表中的任何值                                       |
| [NOT] BETWEEN n AND m | 相当于`[Not] >= n` 且 <`= y`。                               |
| [NOT] EXISTS          | 如果子查询返回至少一行，则返回`true`                         |
| IS [NOT] NULL         | 测试`NULL`的值                                               |
